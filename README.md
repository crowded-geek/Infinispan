# Infinispan
for GCI

For starting the project make sure you have maven and infinispan server installed.
- Download or clone the repository
- Open Terminal
- `cd infinispan`
-  start infinispan server
- `mvn clean package`
- `java -jar target/infispan-1.0-SNAPSHOT.jar`

## Voila! It works!
